# 🧮 Advanced Calculator

A scientific calculator with a real GUI, full keyboard support, and a dark mode — built entirely on Python's standard library. No dependencies, no bloat.

## Features
- Standard arithmetic + scientific functions: `√`, `sin`, `cos`, `tan`, `log`, `eˣ`, `π`
- Full keyboard binding — type and calculate without touching the mouse
- One-click **dark mode** toggle
- Backspace, clear, and error-safe evaluation

## Tech Stack
`Python` · `Tkinter` · `Math` (stdlib only — zero pip installs required)

## Run It
```bash
python advanced_calculator.py
```

## Why It's Solid
- Pure standard library — runs anywhere Python runs
- Responsive grid-based button layout
- Trig functions operate in degrees for intuitive everyday use
